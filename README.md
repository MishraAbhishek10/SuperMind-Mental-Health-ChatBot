# 🧠 SuperMind Mental-Health ChatBot 🩺

SuperMind is an **empathetic mental health chatbot** that provides a safe space to talk, relax, and feel supported. It supports **text and voice interaction**, detects your mood, and suggests calming activities to help improve emotional wellbeing.

---

## **Features**

- 💬 **Text & Voice Chat:** Talk to SuperMind using your keyboard or microphone.  
- 🧠 **Mood Detection:** Automatically detects your emotional state (Happy, Sad, Stressed, Anxious, Neutral).  
- 🧘 **Calming Activities:** Provides personalized activity suggestions based on your mood.  
- 🌱 **Empathetic Responses:** Friendly and concise responses generated by Groq LLM.  
- 🎵 **Voice Output:** Bot responses can be read aloud using TTS (`pyttsx3`).  
- 📜 **Session Management:** Maintains chat history for ongoing conversations.  

---

## **Tech Stack**
- Frontend: Streamlit
- Backend: Python
- LLM: Groq LLaMA 3.1
- Voice Recognition: speech_recognition
- Text-to-Speech: pyttsx3
- Environment Management: .env
